Symlinks notes for my windows environment:

1. To generate my list of symlinks:
dir /A:L /S

2. Favorites (Explorer Side Bar)
$userprofile/Links

3. Libraries
$userprofile/AppData/Roaming/Microsoft/Windows/Libraries

4. Themes
$userprofile/AppData/Local/Microsoft/Windows/Themes

5. To generate windows home structure:
tree /A > windows.homedirectory.txt
