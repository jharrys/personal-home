Symlinks notes for my windows environment:

1. To generate my list of symlinks:
dir /A:L /S

2. Favorites (Explorer Side Bar)
$userprofile/Links

3. Libraries
$userprofile/AppData/Roaming/Microsoft/Windows/Libraries

4. Themes
$userprofile/AppData/Local/Microsoft/Windows/Themes

5. To generate windows home structure:
tree /A > windows.homedirectory.txt

6. To generate windows.filetypes.txt
METHOD 1 GUI
Open from SysInternals FileTypesMan.exe > View > HTML Reports (All)

METHOD 2 Command Line
C:\Users\lpjharri\Documents\Sysinternal Tools\filetypesman-x64\FileTypesMan.exe /scomma windows.filetypes.txt

METHOD 3 Command Line
C:\Users\lpjharri\Documents\Sysinternal Tools\filetypesman-x64\FileTypesMan.exe /shtml windows.filetypes.txt
