# Windowing
* :only - kill all other windows
* map <F5> <C-W>_<C-W><Bar> - map to make the current window as large as possible without closing other windows

# Copy/Paste System clipboard on Mac OS X
* select using v or V, then type :w !pbcopy
* paste from clipboard :r !pbpaste
* to copy the whole file do :w% !pbcopy

# Identify current filetype
* :set ft?

# To see current indent settings and where they were set
* :verbose set ai? cin? cink? cino? si? inde? indk?

# To prevent auto indenting for the current working file
* :setlocal noautoindent
* :setlocal nocindent
* :setlocal nosmartindent
* :setlocal indentexpr=
* to do all above in a single line do - ":setl noai nocin nosi inde="
* or map to F8 - ":nnoremap <F8> :setl noai nocin nosi inde=<CR>"

# Triggering filetype for the current file
* :set ft=php (to set for PHP)

# Search and replace word under cursor
* Place cursor on _foo_
* Press _*_ to search for the next occurrence
* Type _ciw_ (change inner word) then _bar_ then press Escape
* Press _n_ (move to next occurrence) then _._ (repeat change)
* Repeat last step
* Turn off highlighting - :noh, :nohlsearch, set hlsearch!, nnoremap <F3> :set hlsearch!<CR>
* 

# Format/Cleanup/Tidy
* gg=G : format whole buffer
* v : go into visual mode
    * = : tidy up highlighted area in visual mode
    
# Indenting
* >>, <<: indent or unindent current line
* >,<: indent or unindent objects, motions or visual selections
* Ctrl-T, Ctrl-D: indent or unindent in insert mode
* == in normal mode to indent current line, = for objects
* gg=G: gg move to the beginning, = indent, G to the end
* :>, :< takes a range to indent (i.e., :12,20>>> : indents lines 12 to 20 three times)

# visual
* gv to reselect the selected lines from before
