## Defining a Root Context in a Spring Boot App

* By default the app will use the context path of “/”
* To change this, the modification goes into the properties (i.e., application.properties)
    
        server.contextPath=/myApp
        
* YAML-based config

        server:
            contextPath:/myApp
            
* Programmatically

        @Component
        public class SomeBean implements EmbeddedServletContainerCustomizer {
            
            @Override
            public void customize(ConfigurableEmbeddedServletContainer container) {
                container.setContextPath(“/myApp”);
            }
        }
