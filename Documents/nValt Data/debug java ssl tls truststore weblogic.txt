# Adding debug options to java
* -Dssl.debug=true
* -Djavax.net.debug=all
* -Djavax.net.debug=ssl

# For Weblogic instances
* -Dweblogic.StdoutDebugEnabled=true
* When it is stated to use the administration port, that is an option in weblogic console to use the administration port .... and it always uses SSL. In HWCIR that is the console port 9100 or 9200.
* Bundling jar files for java clients connecting to weblogic console:
    * wlthint3client.jar
    * wlclient.jar
    * wljmxclient.jar

# For the trustStore
* -Djavax.net.ssl.trustStore=/path/to/DemoTrust.jks
* -Djava.net.ssl.trustStorePassword=PASSWORD
* DemoTrust lives in weblogic under MW_HOME/wlserver/server/lib/DemoTrust.jks
* Other certs live under $JAVA_HOME/jre/lib/security/cacerts
