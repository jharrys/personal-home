# Bundle

The homebrew utility for dumping all your installs into __Brewfile__ so that it can be recreated on a new mac or restored from backup.

See https://github.com/Homebrew/homebrew-bundle

## commands:

* to execute your set of installs stored in the file “Brewfile” execute

    ```
    brew bundle
    ```

* to create your Brewfile execute - will create a new Brewfile; —force will overwrite

    ```
    brew bundle dump
    ```
    
* to remove all brews not listed in Brewfile

    ```
    brew bundle cleanup
    ```

* to check to see if you Brewfile needs updating run

    ```
    brew bundle check
    ```
