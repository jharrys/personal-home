# Legend
⌘ = Command     
⌥ = ALT  
⌃ = Control    
⇧ = Shift  
⎋ = Esc   
 ⇥ = Tab   
 ⏏ = Eject    
 ⌫ = Back space (back delete)   
 ⌦ = Forward delete   
 ⏎ = Enter   
 ↖︎ = Home
 ↘︎ = End
 ⇞ = Page Up
 ⇟ = Page Down

# Zsh Shortcuts

|⌥-P||call last cmd run|
|⌥-D||delete word after cursor|

|⌥-F||move forward one word|
|⌥-U||transform word after cursor to uppercase|
|⌥-L||transform word after cursor to lowercase|
