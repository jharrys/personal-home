# diskutil

Used to modify, verify and repair local disks.

`list <device>`
: List disks, including internal and external disks, whole disks and partitions, and various kinds of virtual or offline disks.

`info <device> | -all`
: Get detailed information about a specific whole disk or partition

`umount | umount [force] <device>`
: Unmount a single volume.

`unmountDisk | umountDisk [force] <device>`
: Unmount **all** volumes on the given partition map; that is, an unmount is attempted on the directly-mountable volume, if any, on each of the whole diskâ€™s partitions.

`eject <device>`
: Eject a disk.

`verifyVolume <device>`
: Verify the file system data structores of a volume. The appropriate fsck program is executed and the volume is left mounted or unmounted as it was before the command.

`repairVolume <device>`
: Repair the file system data structures of a volume. The appropriate fsck program is executed and the volume is left mounted or unmounted as it was before the command.

**verifyDisk** and **repairDisk** are available but for the whole disk.
