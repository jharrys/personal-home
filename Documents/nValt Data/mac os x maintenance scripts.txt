# See man periodic.conf
# See /etc/periodic/daily,monthly,weekly
### As of El Capitan (10.11.x)

1. In periodic.conf, you can modifiy “daily_output”, “weekly_output”, or “monthly_output” with a username|email address or a file path. It will either email it to the local user/email address or save it to the file path. If saved to file path newsyslog will rotate the files.
2. You can also add your own scripts /etc/periodic/{period}/XXX.myscript.sh
