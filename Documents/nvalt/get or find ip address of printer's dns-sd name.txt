1. Use dns-sd to browse for printer service types

        $ dns-sd -B _printer._tcp
        << LIST OF PRINTERS: Domain …. Service Type …. Instance Name >>

2. Lookup specific service instance to get details and interface reference

        $ dns-sd -L “Instance Name” [space] _printer._tcp. [space] domain.
    
        << OUTPUT: xxxx can be reached at abchex—morehex-.local.:80 >>

3. Get IPv4 address info from reached at reference

        $ dns-sd -G v4 abchex—morehex-.local.
        
        << OUTPUT: Hostname Address (IPv4) TTL >>
        
        
OUTPUT example

```
14:38:28.443  RDV662F4E\032(BCB181662F4E)._ipp._tcp.local. can be reached at rdv662f4e--bcb181662f4e-.local.:631 (interface 13)
```

**(interface 13)** can be found using the command `netstat -i`
