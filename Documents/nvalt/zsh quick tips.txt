# ZSH tricks
|key/cmd||description|os|
|------|---------------|
|__ctrl-u, ctrl-y__| |*dismiss current command, then bring it back*|unix|
|__tab__| |*command under mac to open tab*|mac|
|__pfs__| | |mac|
|__pfd__| | |mac|
|__cdf__| | |mac|
|__quick-look__| | |mac|
|__man-preview__| | |mac|
|__bindkey list__| |*list key bindings*|unix|
|__bindkey ‘\e#’ pound-insert__| |*how to bind ESC # to comment out lines*|unix| 
|__!:2__| |*3rd arg from prev cmd (use tab to show it)*|unix|
|__^[#__| |*I have this bindkey to comment out current line; it will also uncomment and execute if already commented*|unix|
|__!6167:s/old/new/__| |*replace old with new on cmd 6167*|unix|
|__!?previouscmd?:s/old/new/__| |*you can use TAB to expand ?previouscommand?*|unix|
|__^Q or ^q__| |*will push line to buffer stack, remove it from current line, lets you run something else and then pops the line back to run on next command*|unix|
|__^A or ^a__| |*executes current line and yanks back the command to run next*|unix|
|__^O or ^o__| |*executes command and then yanks next command from history (best when running a prev command from history)*|unix|
