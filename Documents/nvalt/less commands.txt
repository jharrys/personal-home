# cool less commands

* esc-F   : like F, but stops if it finds the last search pattern
* esc-space   : scrolls forward a full screen even if EOF
* F       : like tail -f, but just repaints the screen to get more
* R       : repaints the screen
* &pattern    : display only the matching lines, not all
* s filename  : save input to filename if you're paging a pipe

#### search options - /pattern or ?pattern - place options at start of pattern
* !       : negate pattern
* \*       : search through all files
* @       : start search of all files at the beginning of file list
* ^K      : highlight searched text, but don't move to it
* ^R      : don't interpret regular expressions

#### &pattern
* !       : negate pattern
* ^R      : don't interpret regular expressions

#### file manipulation
* :n      : next file
* :p      : previous file
* :e file2    : open file2
* :x      : examine first file in command line
* :d      : remove current file from the list
* !       : invokes shell
* !(%,#)  : % name of current file, # name of previous file

#### other useful less commands
* 10j     : 10 lines forward
* 10k     : 10 lines backward
* ctrl-g      : status, current file with line, byte and stats
* ma      : mark the current position with letter 'a'
* 'a      : go to the marked position 'a'

#### options
* environment can set default options by:
* LESS="-options"; export LESS
* -S      : chop lines
* -s      : squeeze blank lines
* -i      : ignore case in searches
* -N      : keep line numbers
* -n      : suppress line numbers
