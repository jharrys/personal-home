**⌥⌘/** :

    popup of recent dirs for server you are on
    
**⇧⌘A** : 

    select the output of last cmd
    
**⌘/** :

    locate cursor

**⌥ - click** :

    to move cursor and avoid sending up/down arrow 
    best when shell integration is installed
    use in vim in normal mode will move cursor!
    
**^⌘ - Arrow Key** :

    Resize split panes with the keyboard
    
**Edit > Paste Special > Paste File Base64-Encoded** :

    Easy uploads of binary files to server.
    Use ‘base64 -D’ to decode (linux) ‘base64 -d’
