# Intellij hints

## completion types
* basic completion (c-space) - up to 3x
    * x2 can be gone directly by using **c-op-space**
* smart completion (c-sh-space)
* post-fix completion (cmd-j)
* statement completion (cmd-sh-enter) - completes suggestions & stmts
* hippie completion (op-/) - completes words & cycles each time / pressed
* go to definition (op-space) - pops up source for definition
